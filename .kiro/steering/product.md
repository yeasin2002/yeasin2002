# Product Overview

Sleek Portfolio is a modern, responsive portfolio website template designed for developers and creators to showcase their work, blog posts, and professional experience.

## Core Features

- Personal portfolio with hero section, about, and work experience timeline
- Blog system with MDX support for rich content
- Project showcase with detailed project pages
- Contact form with Telegram bot integration
- Setup/gear section to display development tools
- Dark/light theme support
- SEO optimized with customizable metadata
- Privacy-focused analytics via Umami

## Target Users

Developers, designers, and creators who want a professional portfolio site with minimal setup and easy content management through MDX files.

## Key Integrations

- Telegram bot for contact form notifications
- Gemini API for AI-powered features
- Umami Analytics for privacy-focused web analytics
- Cal.com for scheduling integration
