# Product Overview

Personal portfolio and blog website for Md Kawsar Islam Yeasin (yeasin2002), a full-stack web developer from Bangladesh.

## Core Features

- **Portfolio Homepage**: Hero section, skills showcase, project gallery, and contact form
- **Blog System**: MDX-powered blog with search functionality using Fumadocs
- **Contact Form**: Server action-based email sending via Resend

## Target Audience

Potential clients, employers, and collaborators looking to learn about the developer's skills and projects.
